A simple plugin to add Google Analytics code to your website.

== Description ==

A simple and easy solution to add Google Analytics code to your WordPress site.

== Installation ==

1. Upload `google-analytics-7` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your Dashboard and navigate to settings > Google Analytics 7.
4. Put the tracking code Google gave you in there.
